Graph:
A graph is a non linear DS, its a collection of nodes and edges.

How to represent a graph:
1. Adjacency Matrix: A 2D array where the value at row i and column j indicates the presence of an edge between nodes i and j.
2. Adjacency List: A list where each index represents a node and contains a list of its adjacent nodes.

Example graph:
0---2---3
|     \ |
4---1---5

This graph can be represented as an adjacency matrix or an adjacency list. For example, the adjacency matrix for this graph would be:

   0 1 2 3 4 5
0  0 0 1 0 1 0
1  0 0 0 0 1 1
2  1 0 0 1 0 0
3  0 0 1 0 0 1
4  1 1 0 0 0 0
5  0 1 1 1 0 0

The adjacency list for this graph would be:
0: [2, 4]
1: [4, 5]
2: [0, 3,5]
3: [2, 5]
4: [0, 1]
5: [1,2, 3]



BFS:
BFS (Breadth First Search) is a graph traversal algorithm that explores the graph level by level. It uses a queue data structure to keep track of the nodes to be explored next.

Ex:
0 --1 ---2
|\
3--4

0 to 1,3,4.
1 to 0,2
2 to 1
3 to 0,4
4 to 0,3


visited: [F,F,F,F,F]
Starting from node 0, the BFS traversal would be:
1. Enqueue node 0 and mark it as visited: visited = [T,F,F,F,F], queue = [0]
2. Dequeue node 0 and explore its neighbors: visited = [T,T,T,F,T], queue = [1,3,4]
3. Dequeue node 1 and explore its neighbors: visited = [T,T,T,F,T], queue = [3,4,2]
4. Dequeue node 3 and explore its neighbors: visited = [T,T,T,T,T], queue = [4,2]
5. Dequeue node 4 and explore its neighbors: visited = [T,T,T,T,T], queue = [2]
6. Dequeue node 2 and explore its neighbors: visited = [T,T,T,T,T], queue = []
